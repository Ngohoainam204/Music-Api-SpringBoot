# 🎵 Music API - Spring Boot

A simple RESTful Music API built with Spring Boot.

## Features

- ✅ Manage Songs, Artists, Albums
- ✅ CRUD operations
- ✅ DTO + Mapper (MapStruct)
- ✅ H2/MySQL Database
- ✅ REST API with JSON

## Technologies

- Java 17+
- Spring Boot 3
- Spring Data JPA
- MapStruct
- SQL Server / H2
